# ALPHANIX Shop Tools & Build Specs

## Screen Drying Rack (Emulsion Drying)
**Purpose:** Horizontal, dark-room drying for 20" x 24" screens.
**Screens:** 20" x 24" (Standard OD).

### Materials List (Home Depot / Local Hardware)
- **Wood:** 1" x 2" x 8' Millstead FJ Spruce KD (or similar).
- **Fasteners:** 1-1/4" Wood Screws.
- **Tools:** Black & Decker 20V Drill, 1/8" drill bit (for pre-drilling to prevent splitting).

### Cut List & Build (Modular 5-Slot Design)
To hold 20" x 24" screens horizontally:
- **Vertical Posts:** 4 pieces @ 36" (Extra height to accommodate UV light housing).
- **Horizontal Rails (Support):** 10 pieces @ 22" (Screen slide-in rails).
- **Cross-Braces (Width):** 4 pieces @ 18" (Front/back connection).
- **Top Mounting Rails:** 2 pieces @ 18" (For mounting the UV light at the top).

### Build Strategy (Combo Unit)
1. **The Ladder:** Build two "ladders" with rails spaced 4" apart, starting from the bottom. Leave the top 12-14" open for the UV light "cone".
2. **The Box:** Connect with cross-braces.
3. **The UV Top:** Mount the UV LED floodlight at the very top, facing down.
4. **The Glass/Weight:** When burning, place the screen on the top rail with a piece of glass on top to press the transparency flat.
5. **The Light Seal:** Cover the sides with black plastic. Use a "door" flap for the screen slots.
